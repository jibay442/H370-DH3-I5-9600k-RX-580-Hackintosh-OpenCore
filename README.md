This is my EFI Folder for Hackintosh for this config :
-Motherboard : Gigabyte H370-DH3
-CPU : Intel Core I5-9600k
-GPU : Sapphire AMD Radeon RX 580 8 Go
-RAM : 32GB DDR4 2666Mhz
